替换版本
```sh
#添加版本到update-alternatives
sudo update-alternatives --install /usr/bin/clang clang /usr/bin/clang-12 100
sudo update-alternatives --install /usr/bin/clang++ clang++ /usr/bin/clang++-12 100

#选择版本
sudo update-alternatives --config clang


```

#下载rustc
